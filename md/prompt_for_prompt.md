## Prompt for prompt

Создай промпт для создания презентации для продукции фирмы ООО «ВМ-Торг» которая является агентом по продвижению и продаже мебельной продукции, изготовляемой по индивидуальным заказам.
- Вся необходимая текстовая информация находится в приложенных DOCX и PDF файлах.
- Образцы продукции находятся в приложенных image файлах, образцы относятся к следующим категориям:
  - столы
  - кабинеты
  - залы
  - музеи
  - лестницы
  - мягкая мебель
- Упор в презентации необходимо сделать именно на уникальности данной продукции и индивидуальному подходу для каждого клиента.
  - Стиль презентации - дружелюбный, не излишне деловой, цепляющий взгляд.
  - Цель презентации - заинтересовать потенциального клиента индивидуальным подходом в изготовлении.
- Ни в коем случае не меняй aspect ratio исходных фото при вставке их в презентацию.
